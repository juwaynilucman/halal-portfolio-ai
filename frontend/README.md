# Frontend — React Dashboard

Placeholder for the React + Recharts dashboard.

Planned components:
- **AllocationChart** — sector/stock breakdown (pie + bar)
- **PurificationTracker** — quarterly charity due per position
- **RedistributionHistory** — buy/sell log with delta fractions
- **ChatPanel** — RAG Q&A interface with streaming responses
- **PortfolioSummary** — total value, ETF split, halal coverage %

Tech stack: React 18, Recharts, TanStack Query, Tailwind CSS.
